Mediabox MCP es un stack de servidor multimedia autoalojado que combina las mejores herramientas de codigo abierto en un solo despliegue cohesivo, y las expone todas a traves de un servidor MCP (Model Context Protocol) potenciado por inteligencia artificial.

## Que es Mediabox MCP?

En esencia, Mediabox MCP agrupa los siguientes servicios en un unico stack de Docker Compose:

- **Jellyfin** -- Servidor de streaming multimedia (peliculas, series de TV, musica)
- **Sonarr** -- Gestion y descarga automatizada de series de TV
- **Radarr** -- Gestion y descarga automatizada de peliculas
- **qBittorrent** -- Cliente BitTorrent para descargas de contenido multimedia
- **PyLoad** -- Gestor de descargas directas
- **Prowlarr** -- Administrador de indexadores para Sonarr y Radarr
- **FlareSolverr** -- Servidor proxy para evadir la proteccion de Cloudflare en sitios de indexadores

Sobre este stack, Mediabox MCP proporciona un **servidor MCP con 30 herramientas de IA** que te permiten gestionar toda tu biblioteca multimedia mediante lenguaje natural -- buscar contenido, iniciar descargas, monitorear progreso, administrar bibliotecas y mucho mas.

## Para quien es?

Mediabox MCP esta disenado para:

- **Entusiastas del multimedia en casa** que desean un servidor multimedia completamente automatizado con configuracion minima
- **Usuarios orientados a IA** que quieren gestionar su biblioteca multimedia a traves de clientes de IA conversacional (Claude, ChatGPT, etc.)
- **Autoalojadores** que valoran la privacidad y el control sobre su infraestructura multimedia
- **Desarrolladores** que desean extender o integrar la gestion multimedia en sus propias herramientas

## Caracteristicas Clave

- **Configuracion con un solo comando** -- Pon todo el stack en funcionamiento con `npx create-mediabox`
- **30 herramientas de IA** -- Busca, descarga, monitorea, importa, reescanea y gestiona contenido multimedia a traves de cualquier cliente compatible con MCP
- **Auto-configuracion** -- Los servicios vienen preconfigurados para comunicarse entre si desde el primer momento
- **Despliegue flexible** -- Ejecuta localmente, en un VPS con proxy inverso, o detras de un Cloudflare Tunnel
- **Seguro por defecto** -- Las claves API, la autenticacion y la red interna se configuran automaticamente
